Actividad de Hardware y Redes

Introducción

El hardware corresponde a las partes físicas de un computador, como la CPU, la memoria RAM, el almacenamiento y la tarjeta madre. Estos componentes permiten procesar información, ejecutar programas y almacenar datos.

Las redes permiten la comunicación entre varios dispositivos. Para ello se utilizan equipos como el router, que dirige el tráfico entre redes; el switch, que conecta dispositivos dentro de una red local; y el punto de acceso, que permite la conexión inalámbrica. Cada dispositivo recibe una dirección IP, la cual lo identifica dentro de la red.
